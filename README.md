# Interactive Calendar Component

## 📌 Description

A responsive and interactive calendar component built using React/Next.js, inspired by a wall calendar design. It supports date range selection, notes management, and theme switching.

## 🚀 Features

* 📅 Day range selection (start & end date)
* 📝 Notes with localStorage persistence
* 🎨 Light/Dark theme toggle
* 📱 Fully responsive design (mobile + desktop)
* ✨ Smooth animations and modern UI

## 🛠 Tech Stack

* React / Next.js
* CSS / Tailwind (or your styling)
* localStorage (for persistence)

## ▶️ How to Run Locally

```bash
npm install
npm run dev
```
